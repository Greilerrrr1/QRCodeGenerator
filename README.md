\# QR Code Generator (GUI)



A Python desktop application that allows users to generate customizable QR codes using a simple graphical interface. Users can enter text or URLs, choose custom colors, and embed a logo in the center of the QR code before saving it as an image.



---



\## Features



\- Generate QR codes from any text or URL  

\- Customize QR code fill color and background color  

\- Add a logo in the center of the QR code (optional)  

\- Save QR codes as PNG image files  

\- Simple and user-friendly graphical interface  



---



\## Technologies Used



\- Python  

\- qrcode library  

\- Pillow (PIL)  

\- Tkinter  



---



\## Installation



Make sure Python 3 is installed on your system. Then install the required dependencies:



```bash

pip install qrcode\[pil] pillow

```



---



\## How to Run



Run the application with:



```bash

python GUI\_QRCode\_Generateor.py

```



A window will open where you can:



1\. Enter text or a URL  

2\. Choose the QR code fill color  

3\. Choose the QR code background color  

4\. (Optional) Select a logo image  

5\. Click \*\*Generate QR Code\*\*  

6\. Choose where to save the QR code  



---



\## Logo Support



If a logo is selected, the program automatically resizes it and places it in the center of the QR code. Transparent PNG logos are supported.



This allows you to create branded or customized QR codes.



---



\## Output



Generated QR codes are saved as `.png` image files and can be used for:



\- Websites  

\- Business cards  

\- Flyers and posters  

\- Marketing materials  

\- Product links
